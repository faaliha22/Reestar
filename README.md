# Reestar - Recipe Website

Reestar is a simple and elegant recipe website built using **HTML and CSS**. This website displays different desserts with cards and includes a contact form for users to get in touch.

---

## Features

- **Navigation Bar**: Includes Home, Recipe, and Contact links for easy navigation.  
- **Recipe Cards**:  
  - Each card displays a dessert with an image, title, description, and "Order Now" button.  
  - All cards have consistent height, width, and image alignment.  
- **Contact Form**:  
  - Inputs for Name, Age, and Email.  
  - Submit button is styled and aligned consistently with the inputs.  
  - Placeholder texts are left-aligned.  

---

## Technologies Used

- **HTML5** – for structuring the website.  
- **CSS3** – for styling and layout.
